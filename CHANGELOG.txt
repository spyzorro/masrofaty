Masrofaty v1.7
- إضافة تقسيم الميزانية إلى فئات مع حد شهري لكل فئة.
- اختيار الفئة أثناء إضافة مصروف كتابة أو فويس.
- أرشيف شهري: كل شهر يتقفل تلقائيًا عند بداية شهر جديد، مع إمكانية قفل الشهر يدويًا.
- تنبيه مصاريف غير طبيعية حسب حد تلقائي أو حد يحدده المستخدم.
- تحسين التصنيف التلقائي الذكي للكلمات مثل Google / Netflix / مطعم / بنزين / STC / صيدلية.
- تذكيرات الاشتراكات قبل الخصم بـ 3 أيام وبـ يوم، بجانب تذكير يوم الخصم.
- إضافة محفظة الكاش: إضافة كاش، صرف كاش، وتعديل الرصيد.
- إضافة اختيار اللغة العربية / English من القائمة.
- مزامنة جوجل تشمل الفئات وأرشيف الشهور والكاش.
