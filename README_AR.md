# مصروفاتي v1.7

تطبيق أندرويد شخصي لإدارة الميزانية والمصاريف والديون والاشتراكات مع قراءة رسائل البنك والمزامنة مع جوجل.

## الجديد في v1.7

- تقسيم الميزانية لفئات: أكل ومشروبات، مواصلات، فواتير، اشتراكات، تحويلات، صحة، كاش، ترفيه، عام.
- أثناء إضافة مصروف كتابة تقدر تختار الفئة يدويًا.
- أرشيف الشهور: كل شهر يتقفل تلقائيًا عند بداية شهر جديد، وتقدر تقفله يدويًا من القائمة.
- تنبيه مصاريف غير طبيعية: لو عملية كبيرة تعدي الحد يظهر تنبيه. الحد ممكن يكون تلقائي أو تحدده من إعدادات التذكير والقفل.
- تصنيف تلقائي ذكي لعمليات البنك والمصاريف.
- تنبيهات الاشتراكات قبل الخصم بـ 3 أيام وبـ يوم ويوم الخصم.
- محفظة الكاش لإضافة وصرف الكاش وتتبعه مع الميزانية.
- اختيار اللغة: العربية أو English.
- المزامنة السحابية تشمل الفئات، الكاش، وأرشيف الشهور.

## طريقة البناء من GitHub Actions

ارفع المشروع على GitHub ثم افتح Actions وشغل Build Android APK. بعد نجاح التشغيل حمل Artifact واستخرج app-debug.apk.

## تحديث ملف update.json

بعد بناء APK جديد ورفعه، حدث ملف update.json:

```json
{
  "latestVersionCode": 7,
  "latestVersionName": "1.7",
  "apkUrl": "رابط ملف APK الجديد",
  "notes": "إضافة ميزانيات الفئات، أرشيف الشهور، تنبيهات المصاريف الكبيرة، محفظة الكاش، واللغة الإنجليزية",
  "dataVersion": "2026-06-27",
  "message": "تحديث جديد في مصروفاتي v1.7"
}
```
