# مصروفاتي

تطبيق أندرويد شخصي لإدارة الميزانية والمصاريف والمديونيات.

## أهم المميزات

- تحديد الميزانية الشهرية والتحكم فيها بالزيادة أو النقصان.
- إضافة مصروف كتابة أو بالفويس.
- قراءة رسائل البنك وتحليلها.
- شراء PoS ومدى يتخصم تلقائيًا.
- شراء الإنترنت يدخل مراجعة قبل الخصم.
- الحوالات الواردة تدخل مراجعة ولا تُسجل إلا باختيار المستخدم.
- سجل عمليات قابل للتعديل والحذف.
- تسجيل الدخول بإيميل Google وحفظ نسخة احتياطية على Firebase / Firestore.
- مركز تحديثات للتطبيق عن طريق ملف update.json.

## الجديد في v1.6

- تذكير قبل الدين بيوم أو بساعتين حسب اختيارك.
- تكرار التذكير لو الشخص متأخر في السداد أو التحصيل.
- زر رسالة واتساب جاهزة بالمبلغ واسم الشخص.
- فلترة سجل العمليات حسب العملة أو التصنيف.
- قسم اشتراكات شهرية متكررة مثل Google وNetflix.
- قفل اختياري للتطبيق بـ PIN أو قفل الجهاز/البصمة لو متاحة.

## الجديد في v1.5

- اسم التطبيق: **مصروفاتي**.
- اختيار العملة من داخل التطبيق: **ريال سعودي** أو **جنيه مصري**.
- قسم **فلوس ليا عند الناس**.
- قسم **ناس ليها فلوس عندي**.
- إضافة تاريخ ووقت لكل دين أو التزام.
- إشعار تلقائي بميعاد السداد أو التحصيل باسم الشخص والمبلغ.
- شاشة الديون بقت تعرض اللي ليك واللي عليك منفصلين.
- الرئيسية بتعرض إجمالي اللي ليك، إجمالي اللي عليك، والمواعيد القريبة.

## صيغة تاريخ التذكير

اكتب التاريخ بهذا الشكل:

```text
27/06/2026 20:30
```

أو:

```text
27/06/2026
```

## البناء من GitHub Actions

ارفع المشروع على GitHub، ثم شغل Workflow:

```text
Build Android APK
```

بعد النجاح حمل ملف APK من قسم Artifacts.
